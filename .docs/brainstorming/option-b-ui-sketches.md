## User Flow Summary

### Happy Path: Task Completion

1. **Sunday**: User receives notification about new group + task assignment
2. **GroupsPage**: User sees active group card + task card at top
3. **Task details**: User taps "Read full" to see complete task description
4. **Complete task**: Group meets during week and completes the task together
5. **Submit report**: User taps "Mark as complete" â†’ uploads photos + writes notes â†’ submits
6. **Pending state**: Task card shows "Under review" status
7. **Admin reviews**: Admin sees submission in review queue â†’ checks photos/notes â†’ approves
8. **Approved**: User sees "Task completed! +10 points" â†’ points added to profile
9. **Feedback**: User separately submits feedback about the meetup experience

### Separation of Concerns

**Task Completion** = Did we do the activity?
- Photos from the task
- Notes about what happened
- Reviewed by admin
- Awards bonus points (+5 text, +10 photo)

**Feedback** = How was the meetup?
- Rating (1-10)
- General experience
- Would meet again
- Auto-approved
- Awards base points (+10)

## Implementation Considerations

**Data structure**:
```typescript
// tasks table
{
  title: string
  description: string
  onlineInstructions: string
  reportInstructions: string
  type: "Activity" | "Conversation" | "Creative" | "Philosophy"
  difficulty: "Easy" | "Medium" | "Hard"
  purpose: "Everyone" | "Romantic" | "Friendship"
  status: "Active" | "Archive" | "Used"
}

// taskAssignments table
{
  groupId: Id<"groups">
  taskId: Id<"tasks">
  assignedAt: number
  completedAt?: number
  completionNotes?: string
  completionPhotos: Id<"_storage">[]
  submittedBy: Id<"participants">
  reviewStatus: "Pending" | "Approved" | "Revision" | "Rejected"
  reviewedAt?: number
  reviewedBy?: Id<"admins">
  reviewComment?: string
  pointsAwarded: number
}

// feedback table (unchanged)
{
  groupId: Id<"groups">
  participantId: Id<"participants">
  rating: number
  textFeedback?: string
  wouldMeetAgain?: "Yes" | "OnlyIfNeeded" | "PreferNew"
  photos?: Id<"_storage">[]
}
```

**Points logic**:
- Base feedback: +10 points (automatic)
- Task text completion: +5 points (after admin approval)
- Task with photos: +10 points (after admin approval)
- Maximum per week: 20 points (feedback + task with photos)
